# The Sword of Little Red Riding Hood
Creative coding visual art using **Processing (Python Mode)**.  
Reimagines the tale in a pop-art aesthetic using generative patterns, loops, and custom functions.

## Project Structure
##how to run
First download processing: https://processing.org
Open 'main.pyde'in **Python mode**
Click **Run**
The sketch will saves a static output to 'output_image.png' via 'saveFrame'

##Feature
This project is an interactive visual art achieved through 'cliking'
